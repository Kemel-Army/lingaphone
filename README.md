# FEMO Platform

Цифровая образовательная экосистема по математике для 1-6 классов.

Технологии: Nuxt 4, Nuxt UI v4, Supabase, TypeScript, pnpm.

## Запуск

```bash
pnpm install
pnpm dev
```

## Основные команды

```bash
pnpm dev
pnpm build
pnpm preview
pnpm lint
pnpm test
pnpm db:migrate
```

## Окружение

Создайте `.env` и заполните:

```bash
SUPABASE_URL=
SUPABASE_KEY=
SUPABASE_SECRET_KEY=
OPENAI_API_KEY=
```
